# 5-a-Side Football Simulator
A console-based football simulation engine built in C++ as a second-year university project.

## Features
- Player creation with performance-based overall ratings
- Team building with calculated team strength
- Match simulation using rating-based outcome logic
- Full league simulation with standings and results tracking

## Tech Stack
C++, OOP (classes: Player, Team, Match, League)

## Grade: First Class
Built as part of second year Advanced Programming module at Sheffield Hallam University.

## How to Run
Compile with any C++ compiler:
g++ main.cpp Player.cpp Team.cpp Match.cpp League.cpp Types.cpp -o football_sim
./football_sim
